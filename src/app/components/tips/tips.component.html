Build and Deploy Angular application to Firebase Project

I create two separate directories on my development workstation

  • Angular application (%%app_root_directory%%)
  • Firebase deployment directory %%firebase_deploy_directory%%

1. Setup Firebase Deployment Directory

NOTE: Linux commands from %%firebase_deploy_directory%%

  • cd %%firebase_deploy_directory%%
  • firebase login
  • Click login http link - login with gmail account
  • firebase init
  • Answer setup questions for Firebase web hosting
  • firebase logout

2. Create Angular application production deployment build

  • cd %%app_root_directory%%
  • ng build --prod
  • cp -r dist/angular-ccja/* %%firebase_deploy_directory%%/dist

3. Deploy Angular application to Firebase web hosting site

  • cd %%firebase_deploy_directory%%
  • firebase login
  • firebase deploy
  • firebase logout

Angular - Create new Angular application

Create new Angular application:

ng new %%angular-app-name%%

where %%angular-app-name%% is name of new Angular project.


Sample - Possible node packages to install

  • npm i -g @angular/core
  • npm i -g @angular/cli
  • npm i -g firebase-tools

GitHub Pages - Simple Static Website Hosting

Create Repository = %%github_username%%.github.io where %%github_username%% is your GitHub username

Clone your %%github_username%%.github.io repository to your workstation
